# Printer Info WinForms

Приложение на C# Windows Forms для получения информации о принтерах и печати.

## Функциональность

- Список всех установленных принтеров
- Поддерживаемые разрешения печати
- Поддерживаемые размеры бумаги
- Печать 5 строк текста
- Печать изображения test.jpg
- Предварительный просмотр
- Диалог выбора принтера

## Интерфейс

- button1 — информация о принтерах
- butPrint — печать
- butPreview — предварительный просмотр
- textBox1 — вывод информации

## Куда можно встроить

- В офисные приложения с печатью
- В диагностические утилиты
- В демонстрацию работы с PrintDocument

## Требования

- Windows
- .NET Framework / .NET Core с WinForms
- Установленные принтеры
- Для печати: test.jpg в папке программы

## Примечание

Изображение должно быть в папке запуска приложения.
